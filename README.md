## 📌 Project Overview

This project is an interactive **IPL Team Comparison Dashboard** developed in **Power BI**. It enables users to compare the performance of any two IPL teams across different seasons and venues using advanced analytics and interactive visualizations.

The dashboard provides insights into:

- Team performance comparison
- Run Rate Analysis
- Economy Rate Analysis
- Batting & Bowling Phase Analysis
- Win Percentage
- Top Performing Players
- Batting Order Analysis
- Head-to-Head Comparison
